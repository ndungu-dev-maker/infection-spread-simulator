# Disease Outbreak Simulation (SIR Model)

This project simulates the spread of an infectious disease using
a simplified SIR epidemiological model.

Population: 1000
Initial infected: 5
Simulation period: 100 days

Technologies used:
- Python
- Matplotlib

The simulation tracks:
- Susceptible population
- Infected individuals
- Recovered individuals

The results are visualized using epidemic curves.
